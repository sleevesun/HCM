# 04_部门级预算申请与审批 - 核心知识总结

## 模块概述
本模块聚焦于公共性质及部门级别预算的线上化录入与审批体验优化，特别是针对移动端审批场景的痛点解决。

## 核心知识点提炼

### 1. 业务痛点与优化目标
- **痛点**：以往业务部门在编制具有“公共池”属性的预算（如：加班费、离职补偿金、外派补贴等）时，由于缺乏整合的录入页面或清晰的审批展示视图，导致流程冗长，审批人在移动端体验极差，难以快速判断是否应当通过。
- **优化方向**：对部门级预算申请进行独立建模与展示优化。

### 2. 移动端审批界面优化 (UX/UI)
- **字段折叠与透出**：在移动端审批卡片上，精简次要信息，透出核心的“申请金额”、“对比去年变化比例”、“部门剩余总包”等关键决策指标。
- **分类录入**：将加班费、补偿金等不同科目的预算申请结构化，支持批量导入或快速填报，降低业务侧手工录入成本。
- **标准化审批路径适配**：
  - **游戏 & 电竞**：HRD -> C&B -> 二级部门负责人 -> HRBP Head -> 业务HR Head -> 1.1部门负责人(如有) -> CEO & 集团HR。
  - **集团职能 & 影视**：HRD -> C&B -> 二级部门负责人 -> 业务HR负责人 -> 1.1部门负责人(如有) -> 集团HR -> 董事长。

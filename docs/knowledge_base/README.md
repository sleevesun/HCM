# HCM 项目知识库 (Knowledge Base)

欢迎来到 HCM (Human Capital Management - 人力预算与 HC 管控) 项目的知识库中心。本知识库聚合了 2024年09月及后续各个阶段（HCM 3.0/3.1/3.2）的核心业务需求、系统设计方案、测试用例与会议纪要。所有文档均已从 PDF 和 DOCX 格式自动化提取为 Markdown，以便于跨系统检索与查阅。

## 📁 目录导览 (Table of Contents)

本知识库按照业务功能模块划分为 6 大领域，点击进入每个领域即可查阅该模块的**核心知识提炼 (`_Module_Summary.md`)** 及底层源文档：

### [00_综合与通用规范](./00_综合与通用规范/_Module_Summary.md)
涵盖项目概况、多系统数据交互日志（如 TA 财务系统对接）、入离职全局控制逻辑及各类重要会议的结论。
- **核心文件**：HCM系统取数及计算逻辑整理、HC控制-入离职相关 等。

### [01_预算编制与控制](./01_预算编制与控制/_Module_Summary.md)
HCM 的核心模块，详细说明了 HCP 3.0 至今的预算编制逻辑、签约金处理、“年初标识”打标方案以及人财一体化（3.1/3.2）的详细控制闭环。
- **核心文件**：【PRD】HCM系统（人财一体_3.1_预算调整与管控）、年初标识&管控校验&按钮优化 等。

### [02_入职与调动校验](./02_入职与调动校验/_Module_Summary.md)
解决“无预算，不发生”的问题，详述在员工准入（入职、调动）环节时，系统如何拦截超编、如何与财务系统（TA）进行工薪预算的双向强校验。
- **核心文件**：HCM-TA+工薪预算数据一致性校验方案、【需求沟通】HCM-TA+工薪预算数据交互 等。

### [03_过渡期HC方案](./03_过渡期HC方案/_Module_Summary.md)
为了应对业务灵活性需求，允许部门在“老员工未走，新员工先入”的交接期进行暂时性超编的解决方案及释放逻辑。
- **核心文件**：【PRD】HCM-过渡期HC方案。

### [04_部门级预算申请与审批](./04_部门级预算申请与审批/_Module_Summary.md)
针对公共属性预算（如加班费、离职补偿金）的线上申请痛点，优化业务部门录入流程及移动端（手机）审批界面的卡片设计。
- **核心文件**：【PRD】HCM_部门级预算申请与移动端审批界面优化。

### [05_系统切换与过渡期执行](./05_系统切换与过渡期执行/_Module_Summary.md)
从旧版 HCP 平滑迁移到新版 HCM 系统的实施计划、数据清洗结转逻辑及上线时间表（Milestones）。
- **核心文件**：HCM_HCP系统切换与过渡期执行方案、HCM系统上线准备文档。

# 工薪预算驾驶舱表格视觉优化方案

## 问题分析

### 当前痛点
1. **列数过多**：单行包含20+列数据（HC当前5列 + 年末目标5列 + 成本3列 + 2027年多列）
2. **视觉疲劳**：大量相似数据列导致用户难以快速定位关键信息
3. **认知负荷高**：表头层级复杂（3-4层），增加理解成本
4. **列区分度低**：相邻列之间缺乏有效的视觉分隔

### 优化目标
- 保持表头结构不变（需求约束）
- 提升列与列之间的视觉区分度
- 降低用户认知负荷
- 保持数据可读性和界面美观性

---

## 方案三：智能高亮与焦点引导

### 设计原理
利用视觉注意力原理，通过智能高亮用户关注的数据列，引导视线流动，降低信息搜索成本。

### 实施步骤

#### 3.1 智能高亮系统
```typescript
// 高亮模式定义
type HighlightMode = 'none' | 'current' | 'target' | 'comparison' | 'custom';

const highlightMode = ref<HighlightMode>('none');

// 高亮规则配置
const highlightRules = {
  current: {
    columns: ['HC-当前-*', '工薪成本-累计已发生'],
    style: 'highlight-current'
  },
  target: {
    columns: ['HC-年末目标-*', '工薪成本-全年预算'],
    style: 'highlight-target'
  },
  comparison: {
    columns: ['HC-当前-*', 'HC-年末目标-*'],
    style: 'highlight-comparison'
  }
};

// 应用高亮
const applyHighlight = (columnKey: string) => {
  if (highlightMode.value === 'none') return '';
  
  const rule = highlightRules[highlightMode.value];
  if (rule.columns.some(pattern => matchPattern(columnKey, pattern))) {
    return rule.style;
  }
  
  return 'dimmed'; // 非高亮列变暗
};
```

#### 3.4 列聚焦功能
```typescript
const isColumnFocusMode = ref(false);
const focusedColumnIndex = ref<number | null>(null);

const handleColumnClick = (columnIndex: number) => {
  if (!isColumnFocusMode.value) return;
  
  if (focusedColumnIndex.value === columnIndex) {
    // 取消聚焦
    focusedColumnIndex.value = null;
  } else {
    // 聚焦新列
    focusedColumnIndex.value = columnIndex;
  }
};

const getColumnClass = (columnIndex: number) => {
  if (isColumnFocusMode.value && focusedColumnIndex.value !== null) {
    return columnIndex === focusedColumnIndex.value 
      ? 'focus-column' 
      : 'dimmed';
  }
  return '';
};
```

#### 3.5 数据异常值高亮
```typescript
// 异常值检测
const detectAnomalies = (value: number, columnKey: string) => {
  const thresholds = {
    'HC-当前-正编': { max: 100, min: 0 },
    '工薪成本-累计已发生': { maxPercentOfBudget: 0.9 },
    // ... 其他阈值配置
  };
  
  const threshold = thresholds[columnKey];
  if (!threshold) return null;
  
  if (threshold.max && value > threshold.max) {
    return { type: 'warning', level: 'high' };
  }
  
  if (threshold.min && value < threshold.min) {
    return { type: 'error', level: 'low' };
  }
  
  if (threshold.maxPercentOfBudget) {
    const budget = getBudgetValue(columnKey);
    const ratio = value / budget;
    if (ratio > threshold.maxPercentOfBudget) {
      return { type: 'alert', level: 'over-budget' };
    }
  }
  
  return null;
};

// 异常值样式
const anomalyStyles = {

};
```

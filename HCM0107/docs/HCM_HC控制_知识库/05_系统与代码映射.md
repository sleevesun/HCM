# 05_系统与代码映射

本篇文档主要面向研发与系统维护人员，将 HCP 3.0 的核心业务逻辑与前端 Vue 系统的具体代码文件进行了关联映射，便于在需求迭代或线上问题排查时快速定位源码。

所有代码链接均相对于当前知识库的相对路径（指向本地 `antigravity/HCM/HCM0107/src/`）。

## 1. 视图页面 (Views)

涵盖预算管控、审批中心及移动端等核心页面的入口文件：

- [**TransitionHCApply.vue**](../../src/views/TransitionHCApply.vue)
  *用途*：HC 过渡期/新增预算申请的表单填写及提交入口页面。关联入职流程、调动流程的发起校验逻辑。
- [**TransitionHCApproval.vue**](../../src/views/TransitionHCApproval.vue)
  *用途*：HC 及预算调整流程的 PC 端审批详情与处理页面。
- [**ApprovalCenter.vue**](../../src/views/ApprovalCenter.vue)
  *用途*：统一的审批中心台，汇总各项（如调薪、HC 预算等）待办审批任务。
- [**BudgetAdjustment.vue**](../../src/views/BudgetAdjustment.vue)
  *用途*：预算调整（不同 HC 类型调拨、项目预算调整、C&B调整等）的综合管理页面。
- [**SalaryBudgetCockpit.vue**](../../src/views/SalaryBudgetCockpit.vue)
  *用途*：人工成本看板/薪资预算驾驶舱，展示部门预实比对、工资与各成本项的明细大盘。
- [**BudgetApprovalMobile.vue**](../../src/views/BudgetApprovalMobile.vue)
  *用途*：预算与 HC 审批的移动端（H5/企微等）适配页面。

## 2. 业务组件 (Components)

核心复用组件，承载了人员展示、部门树以及流程操作的 UI 与交互逻辑：

- [**PersonnelPopover.vue**](../../src/components/budget/PersonnelPopover.vue)
  *用途*：人员信息悬浮卡片组件，在看板或预算明细中展示员工的类型（正式、派遣、兼职等）及发薪成本归属。
- [**BudgetDetailTable.vue**](../../src/components/budget/BudgetDetailTable.vue)
  *用途*：预算明细数据表格，展示 PS 科目映射后的各项薪资结构（工资、社保总额、公积金等）。
- [**DepartmentTreeSelect.vue**](../../src/components/DepartmentTreeSelect.vue)
  *用途*：部门树形下拉选择组件，严格按照 HCP 3.0 定义的四级部门/管控层级进行数据过滤与展示。
- [**FlowReturnButton.vue**](../../src/components/FlowReturnButton.vue)
  *用途*：审批流回退/退回操作按钮组件。

## 3. 状态管理与路由 (Store & Router)

- [**salaryBudget.ts (Store)**](../../src/stores/salaryBudget.ts)
  *用途*：全局管理薪资预算、预实比对数据、看板筛选条件的 Pinia 状态树。
- [**user.ts (Store)**](../../src/stores/user.ts)
  *用途*：全局管理当前登录用户信息及**特殊权限矩阵**（如高管组、HRD、特殊授权白名单等）的状态。
- [**index.ts (Router)**](../../src/router/index.ts)
  *用途*：Vue Router 路由配置表，包含各页面的路径定义与前端权限路由拦截守卫。

## 4. API 与服务 (API & Services)

- [**transitionHcQuotaApi.ts (Mock/API)**](../../src/mocks/transitionHcQuotaApi.ts)
  *用途*：定义了与后端交互获取“HC 空余编制”、“离职替换 HC”及“过渡期预算额度”的核心接口。
- [**budgetApprovalMobileService.ts**](../../src/services/budgetApprovalMobileService.ts)
  *用途*：封装移动端预算审批流程专属的业务请求与数据处理服务。

## 5. 工具类与常量 (Utils & Constants)

- [**getTransitionHcQuota.ts**](../../src/utils/getTransitionHcQuota.ts)
  *用途*：核心计算工具函数。用于在入职、调动发起前，根据人员类型（正式/实习）和预计入职时间，计算并校验当前部门的剩余 HC 及可用预算。
- [**deptStructure.ts**](../../src/constants/deptStructure.ts)
  *用途*：部门组织架构的静态常量及层级定义（如 1.1 级部门规则、-1 层管控规则映射等）。

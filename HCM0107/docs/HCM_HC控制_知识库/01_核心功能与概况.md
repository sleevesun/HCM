# 01_核心功能与概况

## 1. 项目目标与前提

HC Planning 3.0（HCP 3.0）旨在彻底打通人力与财务系统间的数据壁垒，实现**人财一体化**的预算与编制管控。

**两大核心前提**：
1. **建立统一的数据底座**：整合人力与财务的核心数据（人员编制、成本科目），确保双方使用一致的预算数据口径，实现全人员类型、全发薪科目的覆盖。
2. **统一管控维度**：统一人力与财务的组织架构层级划分（如部门层级统一为四级部门、项目层级统一按地区划分），确保对“部门”、“项目”、“人员”等基础维度的定义完全一致。

**应用范围**：
- **人员类型**：正式员工（含试用、返聘）、实习生、劳务派遣、兼职。
- **适用业务**：A股上市公司（游戏、电竞、影视、集团等）。

## 2. 核心功能全景

HCP 3.0 的核心能力围绕**预算**与**编制（HC）**展开，包含以下四大模块：

### 2.1 预算编制
- **细化至 HC 级**：将预算编制颗粒度从部门级细化至 HC（Headcount）级，支持对单个 HC 进行精准的预算编制与管理。
- **多维度支持**：支持“部门 + 项目 + 公司”的多维度预算编制，满足跨部门协作场景下的成本分摊与跟踪需求。
- **平滑过渡支持**：提供 HCP 2.1 到 3.0 的平滑过渡，包含看板功能（立、目、实）以及预算编制、使用、剩余情况的清晰展示。

### 2.2 预算控制与联动
- **多系统联动**：预算编制、调整、执行等数据变更会实时推送至 HR、财务、业务（如 OA、数据中台）等系统，确保信息一致性，消除二次加工。
- **动态触发**：
  - 组织结构变化（如部门合并、拆分）或人员编制变化（岗位调动、离职）发生时，系统自动触发预算的重新分配或释放。
  - 员工入离调等操作自动同步至预算模块，触发对应成本科目的启用、冻结或转移。
- **预实比对**：
  - 支持下钻至任意层级部门（如具体小组）进行比对，不再局限于二级大部门。
  - 新增“工资预算与实际工资占用”的金额比对，与预算管控逻辑强对应。

### 2.3 预算调整
支持在系统内进行多种场景的预算调整申请，包括：
1. 新增 HC 预算。
2. 不同 HC 类型间的预算调拨。
3. 组织架构调整带来的预算转移。
4. C&B（公司层面）的全局调整。
5. 项目级预算调整。

### 2.4 多版本管理
- 记录预算编制的多个版本（如初稿、修订版、终审版）。
- 支持多版本对比与历史版本回溯，确保预算调整全过程透明且留痕。

## 3. 系统架构与交互
- **统一预算出口**：HCP 作为预算填报、调整的源头系统。
- **流程统一管控**：通过系统完成 HC 的填报计算及线上审批。
- **数据对接到 OA**：与 OA 系统的 HC 控制模块（1.0 & 2.0）深度对接，在发进入职、调动流程时进行 HC 与预算的实时双重校验。
- **TA 系统一致性校验**：支持 HCM 与 TA 系统间的数据比对，支持 4 级及以上部门，显示 TA 更新时间，避免时间差问题。

# 03_权限与数据规则

本指南详细说明了 HCM 系统中特殊用户的权限矩阵、不同人员类型（派遣、兼职）的数据范围界定，以及人工成本核算过程中 PS 薪资项与 HCM 财务科目的映射规则。

## 1. 权限配置规则

系统在常规的角色权限（如 HRBP、C&B、HRD）外，针对特定业务线和高管支持团队进行了特殊授权及管理层级定义。

### 1.1 部门级预算维护与 HRD 设置
- 部门级预算维护的默认责任方为 **HRD**。
- **高管薪资组与管理部门**：部分高管被安排在 5 级子部门下，预算及算薪权限需统一上卷至“集团高管办公室”。
- 典型部门 HRD 示例（需在上线前 Review 确认）：
  - 内审部：王娟
  - 全球泛娱乐业务：白陆
  - 高招办：郭大任
  - 集团办公室下属其他部门：刘恋

### 1.2 预算管控层级
- 明确预算管控的部门层级：通常管控到预算汇总部门的 **-1 层**。
- **特例**：直接挂在 1.1 级部门下的 3 级部门，同样管控到 -1 层。

除了正式员工外，系统需纳管劳务派遣与兼职员工的人工成本。需在上线前 Review 这些存量信息的员工类型与薪资标准。

### 2.1 劳务派遣员工（共 47 人，其中 31 人算薪）
- **电竞**：29人（算薪，非多岗）
- **游戏-天智游**：1人（算薪，多岗：一个虚拟公司岗位+一个实体公司岗位）

## 3. 财务科目与人工成本数据映射

为了实现**人财一体**，HCM 系统需将底层发薪系统（PS）中繁杂的薪资项数据，聚合并映射至标准的财务预算科目中。

### 3.1 海外发薪科目映射
针对海外业务的发薪，目前主要将以下 PS 科目映射至 HCM 科目中（其他如所得税等暂不纳入系统成本计算）：

| PS 海外发薪科目 | HCM 对应科目 | 备注 |
| :--- | :--- | :--- |
| 计发工资 | 工资 | |
| 雇主养老金 | 养老保险 | 计入“社保总额”（海外目前仅核算养老） |

### 3.2 境内人工成本看板指标映射（核心分类）
人工成本看板通过对底层薪资项（数据字段）进行分类汇总，形成直观的成本结构。以下为主要薪资项向 HCM 财务科目的映射规则归纳：

#### A. 映射为“工资”科目
包含所有直接发放到员工账户的常规薪资、津贴及扣款项。
- **正向发放**：计发工资 (`pwrd_er_total_pay`)、各项补贴（交通、住房、外派、电话费）、各项奖金（内推奖、工作奖励、专利奖酬）、其他收入（税前/税后）。
- **逆向扣款**：病假扣款 (`pwrd_dd_lv_sick`)、事假/迟到/旷工扣款、住宿/餐费扣款、违规处罚、标准工时差工资等。
- **离职专项**：劳动补偿金 (`pwrd_er_term_comp`) -> 映射为**离职补偿金**子科目。

#### B. 映射为“加班费”科目
- 工作日加班工资 (`pwrd_er_ot150`)
- 双休日加班工资 (`pwrd_er_ot200`)
- 节假日加班工资 (`pwrd_er_ot300`)
- 加班餐补 (`pwrd_er_ot_meal`)
- *注：包含上月的历史加班费补发项。*

#### C. 映射为“社保总额”科目
汇集雇主承担的各类社会保险费用及补缴费用。
- 雇主养老金 (`pwrd_pension_er`)
- 雇主医疗保险 (`pwrd_medical_er`)
- 雇主大额医疗保险 (`pwrd_lami_er`)
- 工伤保险_雇主 (`pwrd_ind_inj_er`)
- 雇主失业保险 (`pwrd_unempl_er`)
- 雇主生育保险 (`pwrd_maternity_er`)
- *注：2024年下半年后，“雇主综合保险”及部分“社保补缴”项不再统计入成本看板。*

#### D. 映射为“公积金总额”科目
- 雇主住房公积金 (`pwrd_house_er`)
- 雇主住房公积金补缴 (`pwrd_d_house_er`)

#### E. 映射为“补充医疗”与“工会费用”科目
- **补充医疗**：雇主补充医疗保险 (`pwrd_sup_med_er`)、雇主意外险、补充医疗-公司。
- **工会费用**：工会费用 (`c_gp_amount5`)、年终奖工会分摊。

*(详细字段对应清单及历史薪资项变更，请参考底层数据映射字典 `人工成本看板指标对照表`。)*

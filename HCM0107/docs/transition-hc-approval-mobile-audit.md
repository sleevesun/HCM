# 过渡期HC审批移动端审计记录

## 性能
- 首屏目标：资源 ≤ 150KB，LCP ≤ 2.5s
- 本地验证方式：`npx lighthouse http://localhost:5174/transition-hc-approval-mobile --preset=desktop`
- 当前实现：移动端骨架屏 + 本地缓存降级 + 接口重试

## 无障碍
- 字号不低于14px
- 主要按钮高度不低于48px
- 主按钮与状态色使用 `#0052D9`
- 区块标题与数据值使用高对比度文字色

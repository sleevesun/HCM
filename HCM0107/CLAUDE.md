# Project Instructions

## General Rules
- **语言 (Language)**: 任何时候，使用中文生成 .md 文档。
- **环境 (Environment)**: 开发系统为 MacOS。

## AI Skills & Standards
- **Spec Review**: 遵循规范文档审查流程，确保完整性、一致性和清晰度。
- **Frontend Design**: 优先使用现代美学、响应式布局、以及流畅的交互动画。
- **Security**: 遵循 Web 前端（Vue/TS）及后端安全最佳实践。
- **Context**: 优先利用 `graphify` 提供的项目知识图谱进行分析。
